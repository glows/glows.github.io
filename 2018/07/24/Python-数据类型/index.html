Python 数据类型

Author: Glow Date: July 24, 2018  16:07:24

Python 中的每个值都有一个数据类型。
在 Python 编程中，一切（万物）皆对象，如同在Unix上一切皆文件。数据类型实际上是类，变量是这些类的实例（对象）。
Python提供多种数据类型来存放数据项集合，主要包括序列（列表list和元组tuple），映射（如字典dict），集合（set）
