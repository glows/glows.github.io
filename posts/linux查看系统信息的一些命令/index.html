linux查看系统信息的一些命令

2018-07-24
3 minute read
操作系统
操作系统

Linux查看系统信息的一些命令及查看已安装软件包的命令（转)

文章分类:操作系统

系统

 1# uname -a               # 查看内核/操作系统/CPU信息
 2
 3# head -n 1 /etc/issue   # 查看操作系统版本
 4
 5# cat /proc/cpuinfo      # 查看CPU信息
 6
 7# hostname               # 查看计算机名
 8
 9# lspci -tv              # 列出所有PCI设备
10
11# lsusb -tv              # 列出所有USB设备
12
13# lsmod                  # 列出加载的内核模块
14
15# env                    # 查看环境变量

资源

 1
 2# free -m                # 查看内存使用量和交换区使用量
 3
 4# df -h                  # 查看各分区使用情况
 5
 6# du -sh <目录名>        # 查看指定目录的大小
 7
 8# grep MemTotal /proc/meminfo   # 查看内存总量
 9
10# grep MemFree /proc/meminfo    # 查看空闲内存量
11
12# uptime                 # 查看系统运行时间、用户数、负载
13
14# cat /proc/loadavg      # 查看系统负载

磁盘和分区

1# mount | column -t      # 查看挂接的分区状态
2
3# fdisk -l               # 查看所有分区
4
5# swapon -s              # 查看所有交换分区
6
7# hdparm -i /dev/hda     # 查看磁盘参数(仅适用于IDE设备)
8
9# dmesg | grep IDE       # 查看启动时IDE设备检测状况

网络

 1# ifconfig               # 查看所有网络接口的属性
 2
 3# iptables -L            # 查看防火墙设置
 4
 5# route -n               # 查看路由表
 6
 7# netstat -lntp          # 查看所有监听端口
 8
 9# netstat -antp          # 查看所有已经建立的连接
10
11# netstat -s             # 查看网络统计信息

进程

1# ps -ef                 # 查看所有进程
2
3# top                    # 实时显示进程状态

用户

 1# w                      # 查看活动用户
 2
 3# id <用户名>            # 查看指定用户信息
 4
 5# last                   # 查看用户登录日志
 6
 7# cut -d: -f1 /etc/passwd   # 查看系统所有用户
 8
 9# cut -d: -f1 /etc/group    # 查看系统所有组
10
11# crontab -l             # 查看当前用户的计划任务

服务

1# chkconfig --list       # 列出所有系统服务
2
3# chkconfig --list | grep on    # 列出所有启动的系统服务

程序

1# rpm -qa                # 查看所有安装的软件包
