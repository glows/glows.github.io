18.把对象的值复制到另一个对象里 - Object.assign()

1const target = { a: 1 };
2
3const source1 = { b: 2 };
4const source2 = { c: 3 };
5
6Object.assign(target, source1, source2);
7target // {a:1, b:2, c:3}

Object.assign方法实行的是浅拷贝，而不是深拷贝。也就是说，如果源对象某个属性的值是对象，那么目标对象拷贝得到的是这个对象的引用。

1const obj1 = {a: {b: 1}};
2const obj2 = Object.assign({}, obj1);
3
4obj1.a.b = 2;
5obj2.a.b // 2
